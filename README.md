# TestRail MCP Server

A Model Context Protocol (MCP) server for integrating with TestRail.

## Features

- **get_case** - Retrieve test case details from TestRail by case ID

## Configuration

Create `.env` file (see `.env.example`):
```
TESTRAIL_INSTANCE_URL=https://your-instance.testrail.io
TESTRAIL_USERNAME=your@email.com
TESTRAIL_API_KEY=your-api-key
```
